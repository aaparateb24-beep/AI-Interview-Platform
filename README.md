
An AI-powered mock interview platform built using React, FastAPI, Groq LLM, and SQLAlchemy.

## Features

* AI-generated interview questions
* AI-based answer evaluation and scoring
* Resume analysis using LLMs
* Interview history tracking
* Real-time feedback generation
* Modern React frontend
* FastAPI backend
* Groq AI integration
* Render deployment

## Tech Stack

### Frontend

* React
* Axios
* React Router

### Backend

* FastAPI
* SQLAlchemy
* Groq API
* Pydantic

### Database

* SQLite

### Deployment

* Render

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
